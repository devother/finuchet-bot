### FinUchetBot
`Expense & Income Tracker`

A Telegram bot for personal financial tracking built in Go using the [Telegram Bot API](https://pkg.go.dev/github.com/go-telegram-bot-api/telegram-bot-api/v5). This bot helps users track expenses, analyze spending patterns, and manage their finances efficiently.
